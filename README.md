# Mi Portafolio Profesional

Este repositorio contiene mi currículum vitae en formato web, desarrollado con HTML, CSS y JavaScript.

- Integración con Font Awesome para iconos

- HTML5
- CSS3
- JavaScript
- Font Awesome
- GitHub Pages
